FAQ

Common questions

Everything you need to know before getting started.

Does it require internet to run? ▼
No. LoadHog runs completely offline on your DZ computer. No internet connection is needed to manifest jumpers, track credits, or generate reports. The only time you need internet is to download the software.
What happens if the computer crashes? ▼
Your data is saved to disk instantly after every action. The app also creates automatic backups every time it opens and closes. You can manually download a backup at any time and save it to a USB drive.
Can we run it on multiple computers? ▼
The app runs on one computer at a time. A second monitor can display the live jumper board by opening a browser on the same computer. The display board updates automatically every 10 seconds.
How does instructor pay work? ▼
When you add a tandem or student to a load, pay is calculated and credited automatically based on the rates you set in Settings. All credits reverse automatically if a slot is removed. Nothing is ever left unbalanced.
What jump types does it support? ▼
LoadHog supports all common jump types — regular licensed jumpers, walk-in tandems, walk-in students, static line progressions, and AFF Cat A through H. AFF Cat A through D automatically show a dual instructor dropdown.
Do we need to install Python or any software? ▼
No. LoadHog comes as a single exe file with everything bundled inside. Just copy the folder to your computer and double click. No installation, no dependencies, no setup.
What if we get a new computer? ▼
Copy the entire app folder to the new computer via USB drive — make sure to include the database file which contains all your data. Double click the exe and everything is right where you left it.
